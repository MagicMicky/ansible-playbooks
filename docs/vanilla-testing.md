# Vanilla Baseline Testing

The `vanilla-testing` branch deploys the vanilla baseline from dotfiles `vanilla-baseline` branch.

This configuration uses:
- No plugins (empty plugin files)
- No completion customization (vanilla zsh defaults)
- No Starship (system default % prompt)
- No fzf/zoxide integration

See: `../_doc/vanilla-roadmap.md` for incremental testing strategy.
